# Front-End Web Development – Video Compilation Page

A simple HTML project that introduces the basics of Front-End Web Development. The page includes short descriptions of HTML, CSS, and JavaScript along with embedded tutorial videos for each topic.

## Features

- Semantic HTML5 structure
- Introduction to Front-End Web Development
- Separate sections for HTML, CSS, and JavaScript
- Embedded YouTube tutorial videos using `iframe`
- Simple and beginner-friendly design

## Technologies Used

- HTML5

## HTML Elements Used

- `<!DOCTYPE html>`
- `<html>`
- `<head>`
- `<meta>`
- `<title>`
- `<body>`
- `<main>`
- `<section>`
- `<h1>`
- `<h2>`
- `<p>`
- `<iframe>`

## Project Purpose

This project demonstrates how to create a structured webpage using semantic HTML and embed external video content with the `iframe` element. It is designed as a beginner-friendly practice project for learning HTML.
