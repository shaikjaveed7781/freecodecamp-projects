XYZ Bookstore

This project is a simple HTML webpage that represents an online bookstore. It displays a list of books with their titles, descriptions, and purchase options. The page also includes buttons to view the shopping cart and proceed to checkout.

Elements Used

- `<h1>`: Displays the main heading of the webpage.
- `<h2>`: Displays the title of each book.
- `<p>`: Provides descriptions and additional information.
- `<div>`: Groups and organizes the content into sections.
- `<button>`: Creates interactive buttons for "Buy Now", "View Cart", and "Checkout".
- `class` and `id` attributes: Used to organize and uniquely identify elements.

Purpose

The purpose of this project is to demonstrate the basic structure of a bookstore webpage using HTML and to practice organizing content with common HTML elements.
