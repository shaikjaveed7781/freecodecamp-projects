# Working with the HTML Audio Element

A simple HTML project that demonstrates how to use the HTML `<audio>` element to embed and play audio files on a webpage.

## Features

- Displays three audio tracks
- Shows song title and artist name
- Built-in audio playback controls
- Loop enabled for continuous playback
- Simple and clean HTML structure

## Technologies Used

- HTML5

## Concepts Covered

- HTML document structure
- Headings and paragraphs
- HTML `<audio>` element
- `src` attribute
- `controls` attribute
- `loop` attribute
- Responsive viewport meta tag
